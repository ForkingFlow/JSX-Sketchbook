# sketchbook

## init
`bun install`
`bun run init`

## create a new pen
`bun run create`
`bun run create -- component -- --name "pen-name" --template standard`

## run a pen
`bun run dev -- <src/pen/index.html>`


## templates
- Standard HTML, CSS, JS
- 3D CSS scene
- Open Props
- Pug and Stylus
- React
- TypeScript
